# Модель Леонтьева — Симулятор отраслевых шоков

Веб-приложение для анализа межотраслевых связей на основе таблиц «Затраты — Выпуск».

## Возможности

- Загрузка файла Excel с таблицами ЗВ
- Гибкая настройка диапазонов данных
- Симулятор шоков конечного спроса (% или млрд тенге)
- Расчёт мультипликаторов ВВП по всем отраслям
- Интерактивные графики
- Экспорт результатов в CSV и Excel

## Установка и запуск

### 1. Установите зависимости
```
pip install -r requirements.txt
```

### 2. Запустите приложение
```
streamlit run app.py
```

### 3. Откройте браузер
Приложение откроется по адресу: http://localhost:8501

## Развёртывание на сервере

### Вариант А — Streamlit Cloud (бесплатно)
1. Разместите папку на GitHub
2. Зайдите на share.streamlit.io
3. Подключите репозиторий — приложение будет доступно по URL

### Вариант Б — Сервер госоргана
1. Установите Python 3.10+ на сервере
2. Скопируйте папку на сервер
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите: `streamlit run app.py --server.port 8080`
5. Настройте nginx как прокси (опционально)
